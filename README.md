
  # Work Hub

  React + Vite 기반의 Work Hub 랜딩 페이지입니다. 애니메이션 로더, 히어로 섹션, 갤러리, 팀 섹션을 포함하며 네비게이션/CTA/팀 카드가 모두 로고와 브랜드 컬러에 맞춰 커스터마이즈되어 있습니다.

  ## 개발/실행
  - 의존성 설치: `npm install`
  - 개발 서버: `npm run dev` 후 http://localhost:3000
  - 빌드: `npm run build`

  ## 주요 위치
  - 로딩 화면: `src/components/Loader.tsx`
  - 히어로/CTA: `src/components/HeroSection.tsx`
  - 팀 카드 및 GitHub 링크: `src/components/TeamSection.tsx`
  - 네비게이션/푸터: `src/components/Navigation.tsx`, `src/App.tsx`
